Password Checker

About the Project

The Password Checker is a tool designed to help users evaluate the strength of their passwords and ensure they are secure against common threats. It performs checks against a database of known compromised passwords and provides feedback on complexity.

This project was built using Python and Flask. It leverages external APIs for breach detection and includes functionality to guide users toward creating stronger passwords.
