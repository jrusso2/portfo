Home Network Audit

Project Overview

This project involved auditing my home network to identify connected devices and assess potential security vulnerabilities. Using tools like nmap and OpenVAS, I conducted comprehensive scans to map devices and detect vulnerabilities. I then addressed the identified issues and documented the entire process for future reference.

What I Learned

Through this project, I deepened my understanding of network security and vulnerability management. I honed my skills in using nmap for device discovery, OpenVAS for vulnerability scanning, and gained insights into securing home networks against potential threats.
